# 8080-go

Implementation of the 8080 based space invaders arcade machine in golang.

## Build

    $ make

## Run

    $ make run

## Keys

    Enter to start
    1 - one player game
    2 - two player game

    Player one:
    q - left
    w - right
    e - fire

    Player two:
    o - left
    p - right
    i - fire
